# Aviso de Contenido y Marcas

Este repositorio contiene una aplicación web fan-made para jugar con canciones de Queen.

## Música y marcas

- Queen, títulos de canciones, nombres de álbumes, marcas y material musical pertenecen a sus titulares correspondientes.
- El proyecto no aloja archivos de audio de Queen.
- `data.js` enlaza a previews públicos remotos de Apple/iTunes mediante URLs externas.
- Si un preview deja de estar disponible, debe actualizarse la URL en `data.js`.

## Imágenes

- `queen_concert_bg_alt.png` es una imagen original procedural generada para este proyecto, sin logos ni personas reales reconocibles.
- El resto de imágenes deben usarse solo si tienes derecho a publicarlas en tu repositorio público.
